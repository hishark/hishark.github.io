## SQL

删病人的时候报错了：

```java
android.database.sqlite.SQLiteConstraintException: FOREIGN KEY constraint failed (Sqlite code 787)
```

看了下发现欠考虑了，病人和病历之间有关联，如果病人已经创建了病历的话，想要删除病人的话，就必须确保把已有的病历都删除掉，没删除掉的话不准删除病人。
