JetBrains Student
为了下Ultimate版本申请一下学生免费授权。 学生申请 选择邮箱申请。 邮箱会收到一封邮件，邮件内的链接科学上网才可以打开。 按提示操作。 打开IDEA，验证页面使用账号密码登陆，会报错Connection Refused，打开终端输入vim /etc/hosts把jetbrains那一行注释掉即可。
